## Project Description

ecommerceWeb_trytry is an e-commerce web application built using ASP.NET Core MVC with .NET 8. It provides a platform for users to register, browse products, add products to their cart, and view seller products. The application aims to simplify the online shopping experience for farmers while providing sellers with tools to manage their products effectively.

## Features

- User registration with roles (Employee, Farmer)
- Product browsing and searching
- Adding products to the cart
- Viewing and managing the cart
- Creating new products (for sellers)
- Viewing seller products with sorting options
- Login Features
- Data Persistence for individual users

## System Functionalities and User Roles

### System Functionalities

- **User Registration**: Users can register with an email, password, and role (Employee or Farmer).
- **Product Management**: Sellers can create new products by providing details such as name, price, image, type, description, produce time, and stock.
- **Product Browsing**: Users can browse and search for products, view product details, and add products to their cart.
- **Cart Management**: Users can view their cart, update quantities, and remove items from the cart.
- **Seller Products**: Users can view products associated with a specific seller and sort them by date or type in ascending or descending order.

### User Roles

- **Employee**: Can register and access the main features of the application, such as browsing products and managing their cart.
- **Farmer**: Can register and access additional features related to product management, such as creating new products and viewing their own products.
